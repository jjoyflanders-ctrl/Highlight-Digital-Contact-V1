# Digital Business Card Repo

## Setup Instructions

1. Upload all files to a new GitHub repository.
2. Add real images to:
   - assets/defaults/wrapper.jpg
   - assets/defaults/building.jpg
   - assets/employees/
3. Edit employees.csv to add or modify employees.
4. Enable GitHub Pages (Settings → Pages → Deploy from branch).

## Photo Rules
- If 'photo' column is blank:
  - Desktop shows wrapper.jpg
  - Mobile shows building.jpg
